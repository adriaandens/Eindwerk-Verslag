Eindwerk-Verslag
================

Verslag van het eindwerk voor Systemen & Netwerken.

### Compileren ###

```bash
latex verslag.tex
bibtex verslag.aux
latex verslag.tex
latex verslag.tex
pdflatex verslag.tex
open verslag.pdf (optioneel)
```

(Steek dat dus in een bash scriptje of in een vim keybinding)
